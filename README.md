lighthouse-client [![Build Status](https://secure.travis-ci.org/jbuck/node-lighthouse-client.png)](http://travis-ci.org/jbuck/node-lighthouse-client)
=================

A NodeJS client for [Lighthouse](http://lighthouseapp.com)

Quick start
-----------

Install using npm: `npm install lighthouse-client`

Configure an instance of the client with some default options:

    var lighthouse = require("lighthouse-client").createClient({
      account: "myaccount", // https://myaccount.lighthouseapp.com
      token: "mytoken" // http://help.lighthouseapp.com/kb/api/how-do-i-get-an-api-token
    });

Do something cool with it, like list the projects on your account:

    lighthouse.listProjects(function(err, projects) {
      projects.forEach(function(project) {
        console.log(project.name + " - " + project.id);
      });
    });

Documentation
-------------

### Required options by all API calls

* `account` - The Lighthouse account you want to get data from. In the url https://myaccount.lighthouseapp.com, it's myaccount.
* `token` - An access token generated by Lighthouse. See http://help.lighthouseapp.com/kb/api/how-do-i-get-an-api-token

### List all projects on an account

`client.listProjects(options, callback)`
